IR Obstacle Avoidance Alarm

Arduino Uno Beginner Project

Project Overview

The IR Obstacle Avoidance Alarm is a simple project designed to demonstrate how to use an infrared sensor with an Arduino Uno. The project detects obstacles using the IR sensor and triggers an alarm (in this case, an LED) when an obstacle is detected within a certain range.

Components Used

  • Arduino Uno
  • IR Obstacle Sensor
  • LED
  • 220Ω Resistor
  • Breadboard
  • Jumper Wires

How It Works

The IR obstacle module has an IR LED(Transmitter) that emits infrared light and a photodiode(Receiver) that detects the reflected light. When an obstacle is detected, the photodiode sends a signal to the Arduino, which then turns on the LED connected to pin 13.

Arduino Code


int irPin = 2;
int ledPin = 13;
int buzzerPin = 8;

void setup() {
  pinMode(irPin, INPUT);
  pinMode(ledPin, OUTPUT);
  pinMode(buzzerPin, OUTPUT);
}

void loop() {
  int irState = digitalRead(irPin);

  if (irState == LOW) {
  digitalWrite(ledPin, HIGH);
  digitalWrite(buzzerPin, HIGH);
  delay(200);
  digitalWrite(buzzerPin, LOW);
  delay(200);
} 
else {
  digitalWrite(ledPin, LOW);
  digitalWrite(buzzerPin, LOW);
}
}
