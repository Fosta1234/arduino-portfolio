LED Blink System

Arduino Uno Beginner Project

Project Overview

This project demonstrates how to control an LED using the Arduino Uno. The LED is connected to digital pin 13 through a 220Ω resistor.

Components Used

  • Arduino Uno
  • LED
  • 220Ω Resistor
  • Breadboard
  • Jumper Wires

How It Works

The Arduino sends a HIGH signal (5V) to turn the LED ON and a LOW signal (0V) to turn it OFF repeatedly using delays.

Arduino Code


void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
