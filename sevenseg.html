Seven segment Display System

Arduino Uno Beginner Project

Project Overview

The seven segment display displays values counting from 1-9.

Components Used

  • Arduino Uno
  • 7 segment display
  • 7 220Ω Resistor
  • Breadboard
  • Jumper Wires

How It Works

A 1-digit-7-segment has segments labeled a-g. Each segment is just an LED. In Common Cathode all negative legs are connected together to GND. A pin is set HIGH to turn ON a segment and LOW to turn OFF a segment.
