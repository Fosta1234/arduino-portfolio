Water Level Detector

Arduino Uno Project

Project Overview

The system reads analog conductivity values from a water level sensor and classifies the water level into three states: Low, Medium and Full.

Components Used

  • Arduino Uno
  • 3 LEDs
  • 3*220Ω Resistor
  • Breadboard
  • Jumper Wires

How It Works

The exposed traces act like many tiny switches.

  • When water touches more lines - conductivity increases.
  • The Sensor outputs a higher analog value.
  • Arduino reads that value and decides the level.
It does not measure depth directly - it measures conductivity across the board.

Arduino Code


int sensorPin = A0;

int redLED = 8;
int yellowLED = 9;
int greenLED = 10;

void setup() {
  Serial.begin(9600);

  pinMode(redLED, OUTPUT);
  pinMode(yellowLED, OUTPUT);
  pinMode(greenLED, OUTPUT);
}

void loop() {
  int waterValue = analogRead(sensorPin);
  Serial.println(waterValue);

  if (waterValue < 200) {
    // LOW
    digitalWrite(redLED, HIGH);
    digitalWrite(yellowLED, LOW);
    digitalWrite(greenLED, LOW);
  }
  else if (waterValue < 550) {
    // MEDIUM
    digitalWrite(redLED, LOW);
    digitalWrite(yellowLED, HIGH);
    digitalWrite(greenLED, LOW);
  }
  else {
    // FULL
    digitalWrite(redLED, LOW);
    digitalWrite(yellowLED, LOW);
    digitalWrite(greenLED, HIGH);
  }

  delay(300);
}
